
# Go Beginner Toolkit – Hello World Example

## Project Overview

This is a simple beginner-friendly toolkit for learning Go (Golang). The project demonstrates how to:

- Install Go on your machine
- Set up a basic Go project
- Run a minimal "Hello, World!" program

This toolkit is designed for absolute beginners who want to start learning Go with a working example and clear documentation.

---

## System Requirements

- Operating System: Windows / Linux / macOS  
- Code Editor: Visual Studio Code  
- Go SDK installed (https://go.dev/dl/)  
- Terminal or PowerShell for running commands  

---

## Project Structure

go-beginner-toolkit/
└── go-hello-world/
├── main.go
└── go.mod


- `main.go` – Contains the Hello World Go program  
- `go.mod` – Go module file for dependency management  

---

## Getting Started

### 1. Clone the Repository (optional if using GitHub)
```bash
git clone <your-repo-url>
cd go-beginner-toolkit/go-hello-world

2. Initialize Go Module (if not already initialized)
go mod init hello

3. Run the Program
go run main.go

Expected Output
Hello, World!

Common Issues & Fixes

Issue: go command not recognized
Fix: Make sure Go is added to the system PATH and restart your terminal or VS Code

Issue: Program does not run
Fix: Always run the program using go run main.go, not by directly opening the .go file
